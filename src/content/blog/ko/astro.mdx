# Astro 란?

Astro는 블로그, 마케팅, 전자 상거래와 같은 콘텐츠 중심의 웹사이트를 생성하기 위해 고안된  프레임워크입니다. Astro는 최소한의 JavaScript를 사용하여 더 빠른 로드 시간을 제공하며, 다양한 프레임워크와 호환되어 개발자들이 선호하는 툴을 사용할 수 있게 합니다. Astro 홈페이지에서 소개되는 특징은 다음과 같습니다.

- **아일랜드 아키텍쳐**: 콘텐츠 중심의 웹사이트에 최적화된 컴포넌트 기반 웹 아키텍처.
- **UI 독립적**: React, Preact, Svelte, Vue, Solid, Lit, HTMX, 웹 컴포넌트 등을 함께 사용할 수 있습니다.
- **서버사이드 렌더링**: 방문자의 디바이스에서 비싼 렌더링을 제거합니다.
- **자바스크립트 없음**: 사이트를 느리게 하는 클라이언트 측 자바스크립트가 적습니다.
- **콘텐츠 컬렉션 지원**: 마크다운 콘텐츠를 구조화하고 타입스크립트 타입 안전성을 제공합니다.
- **Customizable**: Tailwind, MDX 및 수백 가지의 통합 옵션을 선택할 수 있습니다.
